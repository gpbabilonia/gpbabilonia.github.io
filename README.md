## About

This is a static single-page portfolio built with plain HTML and CSS — no frameworks, no dependencies. It covers research projects, work experience, education, milestones, and a side quests section.

## Stack

- HTML5 / CSS3 / Vanilla JS
- Google Fonts — Playfair Display + DM Sans
- Hosted on GitHub Pages
